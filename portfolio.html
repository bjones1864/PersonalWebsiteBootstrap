Portfolio

Some Notable Projects:

OMDB API Display

This is an MVC application that gets data on movies from the OMDB API and displays it to the user. It takes user input and turns that into an API request. It can find either one movie at a time or three. It uses Newtonsoft.Json and is intended for demonstrating how to turn user input into API calls.

Animal Shelter Software

This MVC application gets data from a database on animals, and allows a user to see the available animals and adopt them (removing them from the database) or rehome an animal (adding it to the database). It's intent was to show off the use of EntityFramework for CRUD applications.

Library System Software

This is a C# console app made for my midterm project at Grand Circus Bootcamp. It allows a user to check out books and movies from a library, as well as return books and donate books. The intent was to show off the use of a class, as well as file input/output.

Pig Latin Translator

This is another C# console app made for Grand Circus Bootcamp. It is a Pig Latin translator, that can translate entire paragraphs with punctuation. This shows off some string manipulation, as well as working with arrays
